```mermaid
sequenceDiagram
participant Client as Client
participant PointController as PointController
participant PointService as PointService
participant PointRepository as PointRepository
participant PointHistoryRepository as PointHistoryRepository

Client ->> PointController: 잔액 충전 요청 (userId, amount)
PointController ->> PointService: 충전 요청
PointService ->> PointRepository: 잔액 조회
alt userId가 0 이하 또는 존재X 경우
PointRepository -->> PointService: 사용자 없음
PointService -->> PointController: 사용자 없음 예외 발생
PointController -->> Client: 404 Not Found
else 사용자가 존재하는 경우(트랜잭션)
PointRepository -->> PointService: 잔액 정보 반환
PointService ->> PointService: 잔액 충전
PointService ->> PointRepository: 사용자 잔액 업데이트
PointRepository -->> PointService: 저장 완료
PointService ->> PointHistoryRepository: 충전 내역 저장
PointHistoryRepository -->> PointService: 저장 완료
PointService -->> PointController: 완료 응답
PointController -->> Client: 충전된 잔액 정보 반환
end
Client ->> PointController: 잔액 조회 요청 (userId)
PointController ->> PointService: 조회 요청
PointService ->> PointRepository: 사용자 잔액 조회
alt userId가 0 이하 또는 존재X 경우
PointRepository -->> PointService: 사용자 없음
PointService -->> PointController: 사용자 없음 예외 발생
PointController -->> Client: 404 Not Found 에러 응답
else 사용자가 존재하는 경우
PointRepository -->> PointService: 잔액 정보 반환
PointService -->> PointController: 잔액 정보 응답
PointController -->> Client: 잔액 정보 반환
end
