```mermaid
sequenceDiagram
participant Client
participant Controller as ProductController
participant Service as ProductService
participant Repository as ProductRepository

    Client ->> Controller : 상품 목록 조회 요청
    Controller ->> Service : 요청 전달
    Service ->> Repository : 상품 목록 조회 

    alt 상품이 없을 경우
        Repository -->> Service : 상품 없음 반환(Null)
        Service -->> Controller : 상품 없음 예외 발생
        Controller -->> Client : 404 Not Found
    else 상품이 존재하는 경우
    Repository -->> Service : 상품 정보 반환
    Service -->> Controller : 상품 정보 응답
    Controller -->> Client : 상품 정보 반환
    end
